# Password-Strength-Interactive
Allow users to learn that using multiple character sets and making passwords longer makes the brute force approach of cracking a password to take a long time.

## Installation Steps
#### Prerequisites
  - node
  - npm
#### Installation
- git clone
- npm install
- npm run dev
